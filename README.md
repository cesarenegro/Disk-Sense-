# DiskSense

DiskSense (app name: DiskLense) is a macOS disk cleanup utility built with SwiftUI. It focuses on surfacing space‑hungry and unnecessary files and helping you clean them up quickly.

## Features
- Large file discovery
- Duplicate file detection
- App uninstaller view
- Startup items manager
- Smart/flash clean workflows
- System stats and health overview

## Tech
- SwiftUI (macOS)
- Xcode project

## Getting Started
1. Open `DiskLense.xcodeproj` in Xcode.
2. Select the `DiskLense` scheme.
3. Run the app.
